# University Stream Analytics

## Description
Implementation of a University data processing system using Java Streams.

The system supports:
- Aggregations (sum, average, statistics)
- Filtering and sorting
- Grouping by attributes
- Nested stream operations
- Optional handling
- Reductions and transformations

## Concepts Practiced
- flatMap
- groupingBy & collectors
- Comparator chaining
- IntSummaryStatistics
- Optional
- Immutable transformation
- Functional-style programming
