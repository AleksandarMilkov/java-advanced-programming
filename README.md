# Java Advanced Programming

Welcome to the **Java Advanced Programming** repository! This project contains a curated collection of programming problems and solutions commonly encountered in advanced Java courses, including exams, midterms, and lab exercises. Ideal for practice, learning, and exam preparation.

---

## Repository Structure

The repository is organized by problem type and contains problems like:

- **General OOP Problems:** Focus on encapsulation, inheritance, polymorphism, and class design.

- **Design Patterns:** Demonstrates common design patterns in Java:

- **Streams & Functional Programming:** Utilizes Java Streams, lambda expressions, and functional interfaces. Example: filtering, mapping, and sorting collections efficiently.

- **Collections & Data Structures:** Practice with Java Collections Framework (List, Set, Map), sorting and grouping using custom comparators, and nested/hierarchical collections.

---

## Recommended Usage

- **Practice & Learning:** Strengthen OOP, design patterns, collections, and Streams knowledge.  
- **Exam Preparation:** Review past exam, midterm, and lab problems.  
- **Reference:** Learn common Java techniques and patterns for your projects.
